Admin Panel

Gone are those days of updating the church app with JSON files. Now we use PHP and MySQL. Unzip your adminpanel.zip, you will an install.sql file. Create a MySQL database and import the install.sql file.

Browse for db_conx.php in adminpanel/includes/db_conx.php. Open it and input your
"SERVER_NAME", "USERNAME","PASSWORD" and "DATABASE_NAME".

The Admin Panel is straight foward and have some instructions on the welcome page so we are not going to waste your time on it.

NB: Create a login page for your admin panel, that way, only you will have access to it.
